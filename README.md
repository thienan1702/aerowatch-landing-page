# AeroWatch X1 - Smartwatch Landing Page

A modern smartwatch landing page built with React, Vite, TailwindCSS, and Framer Motion.

## Features

- Responsive Design
- Dark Mode
- Hero Section
- Product Features
- Product Specifications
- Image Gallery
- Customer Reviews
- Newsletter Subscription
- AI Chatbot Assistant
- Scroll Animations
- SEO Optimization

## Technologies

- React
- Vite
- TailwindCSS
- Framer Motion
- Lucide React
- React Icons
- React Hot Toast

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
